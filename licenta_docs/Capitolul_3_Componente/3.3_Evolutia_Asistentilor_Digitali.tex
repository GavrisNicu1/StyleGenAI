\section{Tehnologii de Dezvoltare Modernă}
\label{sec:tehnologii_dezvoltare}

Pentru implementarea unui sistem scalabil și performant, alegerea stivei tehnologice (\textit{tech stack}) este crucială. Proiectul utilizează paradigme moderne de dezvoltare web și mobilă.

\subsection{Dezvoltare Mobilă Multiplatformă: React Native}
React Native, dezvoltat de Meta, permite construirea de aplicații native pentru iOS și Android folosind o singură bază de cod scrisă în JavaScript/TypeScript.
\begin{itemize}
    \item \textbf{Arhitectura Bridge}: Permite codului JavaScript să comunice asincron cu modulele native ale dispozitivului (Cameră, Stocare).
    \item \textbf{Expo Framework}: Un ecosistem de instrumente construit peste React Native care simplifică accesul la API-urile native și procesul de build/deploy. Utilizarea \textit{Expo Router} permite o navigare fluidă bazată pe fișiere, similară cu dezvoltarea web modernă \cite{reactnative_doc}.
\end{itemize}

\subsection{Backend și Microservicii: Python & Flask}
Python este limbajul dominant în ecosistemul de Inteligență Artificială, datorită bibliotecilor precum PyTorch, TensorFlow și NumPy.
\begin{itemize}
    \item \textbf{Flask}: Un micro-framework web minimalist, ideal pentru crearea de API-uri RESTful rapide. Spre deosebire de Django, Flask oferă flexibilitate maximă în structurarea proiectului, permițând integrarea ușoară a modelelor ML \cite{flask_doc}.
    \item \textbf{REST Architecture}: Comunicarea între client și server respectă principiile REST (Representational State Transfer), utilizând verbe HTTP standard (GET, POST, PUT, DELETE) și formatul JSON pentru schimbul de date.
\end{itemize}

\subsection{Persistența și Securitatea Datelor}
\begin{itemize}
    \item \textbf{SQLite}: O bază de date relațională serverless, ideală pentru prototipare rapidă și aplicații mobile, care suportă standardul SQL complet.
    \item \textbf{JWT (JSON Web Tokens)}: Standardul industrial pentru autentificarea stateless. Serverul nu stochează sesiuni, ci validează semnătura criptografică a token-ului trimis de client la fiecare cerere, asigurând o scalabilitate orizontală ușoară.
\end{itemize}

Această combinație de tehnologii (React Native + Flask + SQLite) reprezintă un echilibru optim între viteza de dezvoltare, performanță și capacitatea de integrare a funcționalităților avansate de AI.
