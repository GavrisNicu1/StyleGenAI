\subsection{Structura Bazei de Date}
Baza de date SQLite a fost proiectată conform celei de-a treia forme normale (3NF) pentru a elimina redundanța și a asigura integritatea referențială. Mai jos sunt detaliate tabelele principale și relațiile dintre acestea:

\begin{itemize}
    \item \textbf{users} - Tabelul central pentru gestionarea identității.
    \begin{itemize}
        \item \textbf{id}: INTEGER (PK) - Identificator unic auto-incrementat.
        \item \textbf{public\_id}: UUID - Identificator public securizat folosit în token-urile JWT.
        \item \textbf{email}: VARCHAR(120) - Adresa de email (unică).
        \item \textbf{password\_hash}: VARCHAR(255) - Parola utilizatorului stocată criptat (bcrypt).
        \item \textbf{created\_at}: DATETIME - Data înregistrării contului.
    \end{itemize}
    \textit{Relație:} 'users' are o relație one-to-many cu tabelele 'clothes' și 'outfits'.

    \item \textbf{clothes} - Inventarul digital al garderobei.
    \begin{itemize}
        \item \textbf{id}: INTEGER (PK) - Identificator unic al articolului.
        \item \textbf{user\_id}: INTEGER (FK) - Referință către proprietarul hainei.
        \item \textbf{image\_data}: BLOB/TEXT - Imaginea codificată sau calea către fișier.
        \item \textbf{category}: VARCHAR(50) - Categoria (Top/Bottom).
        \item \textbf{color}: VARCHAR(50) - Culoarea dominantă detectată (ex: 'rosu', 'albastru').
        \item \textbf{usage\_count}: INTEGER - Contor pentru frecvența utilizării.
    \end{itemize}
    
    \item \textbf{outfits} - Istoricul combinațiilor generate.
    \begin{itemize}
        \item \textbf{id}: INTEGER (PK) - Identificator unic.
        \item \textbf{user\_id}: INTEGER (FK) - Proprietarul ținutei.
        \item \textbf{top\_id}: INTEGER (FK) - Referință către piesa superioară.
        \item \textbf{bottom\_id}: INTEGER (FK) - Referință către piesa inferioară.
        \item \textbf{match\_score}: FLOAT - Scorul de compatibilitate vizuală.
        \item \textbf{created\_at}: DATETIME - Momentul generării.
    \end{itemize}
\end{itemize}
